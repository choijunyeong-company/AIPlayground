# Read Handoff

1. 프로젝트 루트 디렉터리에 `handoff` 디렉터리가 존재하는지 확인합니다.
   - 존재하지 않는 경우, 사용자에게 handoff 파일이 위치한 경로를 입력받습니다.

2. handoff 디렉터리 내 파일 목록을 확인합니다.
   - 파일이 없는 경우, 사용자에게 안내하고 종료합니다.
   - 파일이 1개인 경우, 해당 파일을 읽습니다.
   - 파일이 2개 이상인 경우, [파일 선택](#파일-선택)을 진행합니다.

3. 선택된 handoff 파일을 읽고 내용을 파악합니다.

4. 파일 읽기가 완료되면 사용자에게 해당 파일의 삭제 여부를 묻습니다.
   - 삭제를 원하는 경우, 파일을 삭제합니다.
   - 삭제를 원하지 않는 경우, 파일을 유지합니다.

# 파일 선택

handoff 디렉터리에 다수의 파일이 존재하는 경우:

1. 사용자에게 가장 최신 파일을 자동으로 선택할지 묻습니다.
   - 최신 파일 선택을 원하는 경우, 수정 시간 기준 가장 최근 파일을 선택합니다.
   - 직접 선택을 원하는 경우, 파일 목록을 나열하고 사용자가 선택하도록 합니다.

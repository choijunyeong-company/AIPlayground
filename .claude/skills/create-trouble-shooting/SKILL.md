# 문서 카테고리 분류하기

TroubleShooting 디렉터리 내부에 존재하는 디렉터리들은 카테고리를 의미합니다. 
해당 디렉터리들의 이름을 읽고 기존 카테고리의 종류를 파악할 수 있습니다.
이를 바탕으로 작성하려는 문서를 특정 카테고리로 분류하고, 해당 디렉터리에 문서를 위치시킵니다.
적절한 카테고리가 존재하지 않을 경우 새로운 카테고리 디렉터리를 임의로 생성하고, 해당 디렉터리에 문서를 위치시킵니다.

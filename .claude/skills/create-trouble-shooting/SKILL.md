name: create-trouble-shooting
description: 해결한 문제상황에 대해 트러블 슈팅 문서를 작성합니다.
---

1. [템플릿 파일](/Users/nhn/Documents/Development/TroubleShootings/template.md)을 읽고 작성 양식을 정확히 숙지합니다.
2. 템플릿에 맞춰 문서를 작성합니다.
3. [카테고리 리스트 파일](/Users/nhn/Documents/Development/TroubleShootings/categories.txt)을 읽고 작성하려는 트러블 슈팅 내역을 분류합니다. 적절한 카테고리가 기존에 존재하지 않을 경우 임의로 선정합니다.
4. [루트 경로](/Users/nhn/Documents/Development/TroubleShootings)에 선정된 카테고리와 일치하는 디렉터리가 있는지 확인합니다.
5. 디렉터리가 존재한다면, 해당 경로에 문서를 생성하고 없다면 디렉터리를 생성한 후 문서를 해당 디렉터리 내부에 생성합니다.
